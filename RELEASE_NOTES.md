# v3.0.0 Release Notes

v3.0.0 是从零重建的现场验证版，不继承 v1/v2 的业务通过结论。

## 主要变化

- 单入口 `$jw`，不再运行或路由到旧 `jw-*` Skills。
- 完整装入七个九段判断方法，不再用几句话代替课程中的判断过程。
- 五阶段按最早业务瓶颈主动调用方法，并强制分析上下游、30天和90天影响。
- 新增机器可校验的方法资格注册表，区分有效原理、项目假设、工程护栏和退役内容。
- v3 项目运行时强制登记方法、证据、备选、反证、依赖、影响、审批与状态。
- 新增 `doctor`、`capabilities`、v2 只读迁移、配比对账、反馈导出和局部失效。
- 扩充口腔行业研究协议，但不预写任何地域客户心理或真实项目结论。

## 现场验证声明

本版本用于用户真实项目从零测试。它已接受结构、运行时和隔离模型验收，但不宣称已经证明真实获客或成交效果。后续是否进入 v3.1，以真实纠正次数、重复提问、成果可用性、跨阶段关联和业务反馈为准。
