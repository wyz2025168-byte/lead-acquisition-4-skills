## 能做什么

`$jw` 以一个入口完成定位、内容、拍摄、内容配比和发布运营，维护30天目标、90天预判、证据、审批、项目记忆、内容库存和局部失效。它优先交付定位决策、客群矩阵、完整脚本、分镜、拍摄排期、配比和运营卡，而不是只提交验收报告。
