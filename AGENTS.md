# 线索获客 4.0 v4.0.0 仓库规则

- `skills/jw/` 是唯一公开运行真源；旧 `jw-*` 不得恢复、路由或参与测试。
- 对外主链仍是定位、内容、拍摄、内容配比和发布运营；内部必须受 B0–O1 约束。
- 七个方法必须保持九段判断程序和机器资格注册；项目事实与真实结果优先。
- 不得加入原始课程、私有追溯、评测答案、真实客户资料或开发者绝对路径。
- 证据与审批分账；无范围匹配生产批准和执行工具，不得发布、投流、外发或联系客户。
- 本版为 Field Validation Release，不保证流量、咨询或成交。
- 默认校验必须同时通过结构和业务不变量；`--structural-only` 不得用于发行验收。
- 发布前运行完整单元测试、自然评测扫描、脱敏回放、`python3 tools/build-release.py` 和 `python3 tools/validate-package.py`。
