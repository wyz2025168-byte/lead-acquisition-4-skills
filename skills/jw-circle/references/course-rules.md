# 线索圈客｜公开运行规则

- Capability: `C20`
- Public release: `v1.0.0-rc1`
- Provenance: 由私有课程真源编译并完成独立评测；公开版省略来源编号、路径、哈希和内部审批记录。

只把 `origin=COURSE_NEW` 的判断当作课程规则。把 `COURSE_INTERPRETATION` 保留为已批准解释，把 `AI_ENGINEERING` 只用于结构、路由、版本、审批和失败控制。不得用验证案例改写规则。

## 决策规则

| ID | Origin | 编译判断 |
|---|---|---|
| C20-D01 | COURSE_NEW | 从目标人关心的兴趣话题、兴趣表达、兴趣人物或兴趣观点中选择开头圈路线，使正确人继续看。 |
| C20-D02 | COURSE_NEW | 由目标客户对该业务的审美与信任判断定义画面圈，不用创作者个人审美或成本代替。 |
| C20-D03 | COURSE_NEW | 让内容每句话持续对同一类人说，并保持在目标人可理解的认知跨度内。 |
| C20-D04 | COURSE_NEW | 输出目标人可接受且有启发的观点，不追求脱离目标人的抽象绝对正确。 |
| C20-D05 | COURSE_NEW | 检查开头、画面、内容、观点与每一句表达是否始终面向同一批目标人。 |

## 不变量

| ID | Origin | 必须保持 |
|---|---|---|
| C20-I01 | COURSE_NEW | 开头、画面、内容、观点与每一句表达必须面向 C10 已批准的同一批人与同一阶段。 |
| C20-I02 | COURSE_NEW | 开头圈四种圈法是开头圈的子结构，不得被误当成四个主圈。 |
| C20-I03 | COURSE_NEW | 画面圈由目标客户的审美和信任判断定义，不等于创作者个人审美、高成本或视觉炫技。 |
| C20-I04 | AI_ENGINEERING | 愿意看不等于愿意买；C20 不输出购买理由、消费旅程或制作方案。 |
| C20-I05 | COURSE_NEW | 播放量和其他流量指标可观察执行，但不能单独证明圈对人。 |
| C20-I06 | AI_ENGINEERING | 圈客 Brief 必须携带 C10 版本引用、中途换人风险、字段级来源、批准状态、版本与哈希。 |

## 边界

- 不为提高播放量、热度或泛流量扩大或更换 C10 人群
- 不更改 C10 已批准的买单前阶段、价值感与话题边界
- 不用创作者个人审美、高成本或制作便利代替目标人画面判断
- 不创建 C30 购买理由、信任机制或消费旅程
- 不创建 C50 的场景、妆造、设备、机位或灯光方案
