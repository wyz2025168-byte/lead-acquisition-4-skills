# 线索生客｜公开运行规则

- Capability: `C40`
- Public release: `v1.0.0-rc1`
- Provenance: 由私有课程真源编译并完成独立评测；公开版省略来源编号、路径、哈希和内部审批记录。

只把 `origin=COURSE_NEW` 的判断当作课程规则。把 `COURSE_INTERPRETATION` 保留为已批准解释，把 `AI_ENGINEERING` 只用于结构、路由、版本、审批和失败控制。不得用验证案例改写规则。

## 已批准解释

- 理念型表达默认停在方向和价值判断，不默认展开具体解决方法；公开版省略冲突定位与审批编号。

## 边界

- 不修改 C00 定位、C10 人群/阶段或 C30 即时购买理由与旅程角色
- 不把方法、模型、产品功能、临时人设或口号当理念
- 不策划创始人本人不认同或不能长期践行的立场
- 不创建 C50 拍摄、场景、设备、妆造或灯光方案
- 不把 已批准解释 的派生解释伪装成无冲突的 COURSE_NEW 课程规则
- 不使用案例数字或短期爆款替代理念一致性
