# 制作呈现｜公开运行规则

- Capability: `C50`
- Public release: `v1.0.0-rc1`
- Provenance: 由私有课程真源编译并完成独立评测；公开版省略来源编号、路径、哈希和内部审批记录。

只把 `origin=COURSE_NEW` 的判断当作课程规则。把 `COURSE_INTERPRETATION` 保留为已批准解释，把 `AI_ENGINEERING` 只用于结构、路由、版本、审批和失败控制。不得用验证案例改写规则。

## 时效性实施分层

- `COURSE_NEW` 只表示 课程历史资料 在受控文本中确实记录了当时的设备、参数、接口、外链与不完整段落。
- `AI_ENGINEERING` 使用政策要求执行前核验当前兼容性，并且不得推断或补写缺失文本；分别由 `C50-E006` 与 `C50-E007` 强制执行。

## 边界

- 不修改 C10 的目标人群或买单前阶段
- 不修改 C20 的开头、画面、内容或观点语义
- 不创建或重写 C30 的购买理由、信任事实或内容角色
- 不创建或重写 C40 的理念
- 不把更昂贵、更豪华或更高级当作普遍正确的呈现标准
- 不把 课程历史资料 的具体型号、参数、接口、外链或截断说明固化为永久规则
