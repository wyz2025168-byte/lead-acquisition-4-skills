# 发行证据

## 私有编译验证摘要

| 阶段 | 结果 |
|---|---|
| 真源与课程编译 | 已完成来源校验、课程地图、合同冻结与能力编译 |
| 独立能力评测 | 七个课程能力双次运行，汇总 70 / 70 通过 |
| 接口与主链 | 接口装配、顺序、边界和失效传播检查通过 |
| 课程案例回放 | 两个端到端课程案例回放通过 |

私有题目、答案、Rubric、来源编号、文件路径、哈希、审批号和逐决策追溯不随公开包分发。

## 尚未包含

G7 真实项目影子运行尚未启动。因此，本发行包代表“课程能力、接口和课程案例已经离线验证”，不代表真实业务效果承诺。

## 发行变换

公开发行副本没有修改课程规则语义、输入输出 Schema、owner 或业务判断。发行层只增加可移植运行说明、本地完整性校验、统一路由入口，并删除公开运行不需要的私有路径、哈希、评测绑定与内部追溯元数据。
