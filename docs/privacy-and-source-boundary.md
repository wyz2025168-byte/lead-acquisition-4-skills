# 隐私与来源边界

## 发行包包含

- 编译后的课程能力规则
- 不包含私有路径、文件名或源哈希的来源边界证明
- 输入输出 Schema、脱敏运行合同与中性示例
- 只显示用例类型和通过状态的公开评测证明

## 发行包不包含

- 原始 DOCX/PDF 课程文件
- 课程案例的外部真实性背书
- 真实客户资料、真实项目输入、发布内容或业务数据
- 内部审批、RCA、GPT Pro 外审包和操作日志
- 课程来源编号、文件名、私有目录、源文件哈希和逐条证据行号
- 黄金题、expected、Rubric、评分答案、测试 ID 和绑定哈希
- 安装者无法访问的本机绝对路径

`source-map.json` 只声明该能力由私有真源完成编译与验证，不暴露私有追溯结构。安装后的 Skill 只能读取自己的目录。
